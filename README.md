# Comparação de Algoritmos de Busca (TSP)

Este repositório contém a implementação de 5 algoritmos para resolver o Problema do Caixeiro Viajante, desenvolvidos para a disciplina de Inteligência Artificial.

## Algoritmos Implementados
* Busca em Largura (BFS)
* Busca em Profundidade (DFS)
* Busca de Custo Uniforme (UCS)
* Busca Gulosa (Greedy)
* A* (A-Star)

## Como rodar
Execute o arquivo principal:
python main.py
